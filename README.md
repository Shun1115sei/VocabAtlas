# VocabAtlas — 自分用英単語フラッシュカード（個人用）

このリポジトリは、作者が個人的に使うための英単語フラッシュカード集です。GitHub Pagesで公開されていますが、外部利用者向けのサポートや機能追加は想定していません。

## 目的
- 自分の学習進捗に合わせて、範囲別のカードで反復学習するための簡易ツール。

## 公開ページ（参照のみ）
- https://shun1115sei.github.io/VocabAtlas/
※ 公開はしていますが、第三者向けのドキュメント・サポートはありません。

## 使い方（自分用）
1. 上記URLへアクセス。
2. メニューから対象範囲（例：「生物」）を選択。
3. カードをめくりながら確認・復習。

## 編集方法（自分向けメモ）
- 単語の追加・修正は各HTML（例：5000_2-3.html、5000_3-4.html 等）を直接編集。
- メニューやタイトルは index.html で管理。
- 変更後は GitHub Pages の自動デプロイ（Deployments）を確認。

## 技術メモ
- 静的HTMLのみ。ライブラリ未使用。
- ブラウザで動作（Chrome/Firefox/Safari/Edge）。
- オフラインで使う場合はHTMLファイルをローカル保存。

## 対応方針
- Issue/PR は原則対応しません（自分用のため）。
- フィードバックや利用は自由ですが、動作保証・サポートなし。

## ライセンス
- 個人学習用として公開。再配布や改変は自由ですが、クレジット表記を推奨します。

## ファイル構成（例）
- index.html（メニュー）
- 5000_2-3.html／5000_3-4.html／5000_4.html／5000_5.html（範囲別カード）
